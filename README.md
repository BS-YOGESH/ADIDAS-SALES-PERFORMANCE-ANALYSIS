# ADIDAS-SALES-PERFORMANCE-ANALYSIS
Interactive Adidas Sales Dashboard 👟. Analyzing $700M+ in simulated retail data to identify growth opportunities across the US market. Key features: Trend analysis, Regional heatmaps, and Retailer performance metrics. Built with Power BI, Power Query, and Advanced DAX

## 📌 Project Overview
The goal of this project was to analyze Adidas sales dynamics across the United States. I transformed raw transactional data into a high-level executive dashboard to help stakeholders understand which regions and product categories are driving the most profit.

## 🛠️ Technical Implementation
* **Data Source:** Multi-city sales data (Foot Locker, Walmart, Kohl's, etc.).
* **Tool:** Power BI Desktop.
* **Key Skills:** Data Cleaning (Power Query), Relationship Modeling, and Visualization.

## 📈 Top Business Insights
* **Top Product:** Men's Street Footwear leads in total sales volume.
* **Top Region:** The West region shows the highest operating margin (approx. 40%+).
* **Retailer Strategy:** Foot Locker is the primary sales driver, but Walmart shows higher growth potential in specific sub-categories.
